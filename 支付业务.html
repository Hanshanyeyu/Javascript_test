点击支付宝 → 是否混付，混付要跳回支支付页 → 提交301支付服务(支付方式ID，支付金额等) → 进入支付宝web页面,支付宝页面会唤醒支付宝app(混付需要进入支付方式页) → 支付宝支付成功会回跳我们的静态页并带参数，我们通过参数状态来进行页面跳转 点击微信支付 → 是否混付，混付要跳回支支付页 → 【非微信浏览器支付】，提交301支付服务(支付方式ID，支付金额等) → 【非微信浏览器支付】，唤醒微信客户端(唤醒链接带入必要参数) → 【非微信浏览器支付】，微信支付完毕或者退出微信支付后回到我们支付页面点击弹层按钮发送微信支付验证服务，最终根据结果来执行！ → 微信浏览器中支付 ，根据各环境配置的微信appid和链接地址在微信环境中跳转到微信支付页面（带入回跳地址） → 微信浏览器中支付 ，支付完成后会回跳我们微信中转页面处理支付状态 点击百度支付 → 方案同支付宝 点击信用卡支付 → 跳转至信用卡页面 → 用户输入银行卡号 → 服务发送至后台验证 → 获取卡类型和支持银行的卡增量进行信息匹配拿到银行名称(如果和增量配不上就走储蓄卡外列支付即通过支付宝支付)，检查限额 → 回退支付首页填写必须要字段(有效期，预留手机号，证件号码，持卡人，手机验证码)
